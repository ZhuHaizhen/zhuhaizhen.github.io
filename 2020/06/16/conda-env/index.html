在pycharm及jupyter中使用conda虚拟环境
发表于 2020-06-16| 更新于 2022-04-05|学习笔记
字数总计:237|阅读时长: 1 分钟
|阅读量:|评论数:

创建conda虚拟环境

conda create -n selenium python=3.8

此处selenium为虚拟环境的名称

conda activate selenium进入创建的虚拟环境，安装所需软件

conda deactivate退出虚拟环境

conda虚拟环境添加到pycharm

打开pycharm，左上角File-Setting-Project-Project interpreter，右边显示的为当前环境及当前环境下安装的包

点击下图齿轮图标，选择Add...

左侧选择Conda Environment

右侧选择Existing environment，在interpreter下拉列表中选择要添加的虚拟环境

添加后即可切换到新的虚拟环境下工作

conda虚拟环境添加到jupyter

打开jupyter内核所在路径C:\ProgramData\Anaconda3\share\jupyter\kernels

新建一个selenium文件夹

复制python3文件夹下的kernel.json文件到新创建的selenium文件夹下

编辑selenium文件夹下的kernel.json文件，将Python3修改成需要在jupyter中显示的名字

在虚拟环境中安装ipykernel库

conda activate selenium

conda install ipykernel

重启jupyter
