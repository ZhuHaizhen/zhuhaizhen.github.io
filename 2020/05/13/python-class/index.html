类和对象

类是创建对象的模板

  • 使用class关键字创建类，命名一般按照大驼峰规则

  • object是最顶级的父类

  • 一个类可以创建多个实例对象

  • self表示实例对象本身

init魔法方法

  • 常用于属性初始化或赋值

  • 每一个类中都有__init__方法，如果没有定义，python会自动创建

  • 在实例化对象时，__init__方法会自动被调用

私有属性和私有方法

  • 在方法和属性前加__定义

  • 只能在类的内部访问和调用

类方法

  • 使用装饰器@classmethod装饰

  • cls表示类本身

  • 可使用类名.方法名直接调用

类方法可用于修改类属性

静态方法

  • 使用装饰器staticmethod装饰

  • 不默认传入self或cls

  • 可使用类名.方法名直接调用

继承

  • 可以继承自一个父类，也可以继承自不同父类

  • 不同父类中如果有同名的属性和方法，调用时根据class.__mro__中的顺序调用

  • 如父类中的方法不满足需求，可以在子类中重写

多态

在不同的类中定义同名函数，从而使同名的实例方法具备不同的功能
