函数的定义和调用

函数的参数

  • 位置参数

  • 不定长参数

    使用*args定义，传入的是一个元组，可包含任意多个元素

  • 关键字参数

    • 不定长关键字参数：使用**kwargs定义，传入的是一个字典

    • 指定关键字参数：用*分隔位置参数和关键字参数，位置参数在前

    • 如果已有不定长参数，后面的关键字参数不需要*分隔

  • 参数定义的顺序

    必选位置参数、默认位置参数、不定长位置参数、必选关键字参数、默认关键字参数、不定长关键字参数

函数的返回值

  • 多个return语句

    一个函数可以有多个return语句，执行到任意一个时，函数的调用就结束了
